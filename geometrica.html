Explicación Teórica

La distribución geométrica modela el número de intentos necesarios para obtener el primer éxito en una serie de ensayos de Bernoulli independientes.

Parámetros

  • p: Probabilidad de éxito en cada intento (en este caso, la probabilidad de ganar con un boleto).

Fórmulas

  • Función de probabilidad: P(X = k) = (1-p)k-1·p
  • Media (Valor esperado): E[X] = 1/p
  • Varianza: Var(X) = (1-p)/p²

Aplicaciones

Esta distribución es útil para modelar:

  • Intentos hasta el primer éxito en cualquier proceso dicotómico
  • Tiempo de espera hasta que ocurra un evento raro
  • Fiabilidad de sistemas (intentos hasta el primer fallo)
