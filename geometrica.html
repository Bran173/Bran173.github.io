Explicación Teórica

La distribución geométrica modela el número de intentos necesarios para obtener el primer éxito en una serie de ensayos de Bernoulli independientes con probabilidad constante p.

Fórmulas clave

  • Función de probabilidad: P(X = k) = (1-p)k-1·p
  • Media (Valor esperado): E[X] = 1/p
  • Varianza: Var(X) = (1-p)/p²

Mejoras implementadas

  • Precisión estadística: Mayor número de simulaciones posibles (hasta 10,000)
  • Visualización mejorada: Gráfico interactivo con comparación teórica
  • Análisis de error: Cálculo del error relativo entre teoría y simulación
  • Optimización de rendimiento: Uso eficiente de memoria para grandes simulaciones
